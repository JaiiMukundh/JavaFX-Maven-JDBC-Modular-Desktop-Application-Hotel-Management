# Hotel Management System

A desktop application for managing hotel operations built with JavaFX, Maven, and SQLite.

## About

This Hotel Management System is a modular Java application that streamlines hotel operations. It provides a user-friendly interface for managing customers, rooms, bookings, and billing.

## Features

- Customer Management
- Room Management & Inventory
- Booking System
- Billing & Invoicing
- SQLite Database for persistence
- JavaFX UI with CSS styling
- Modular Java architecture (JPMS)

## Tech Stack

- **Java 17+** - Programming language
- **JavaFX** - UI Framework
- **Maven** - Build tool
- **SQLite + JDBC** - Database
- **JPMS** - Module System
